# 📱 Social Media App

A modern full-stack social media solution focused on performance, user experience, and secure authentication.

## ⚡ Quick Start
- **Runtime:** Bun
- **Framework:** Next.js 15 (App Router)
- **Database/Auth:** Supabase
- **UI Library:** shadcn/ui + Tailwind CSS

## 🛠 Installation
```bash
bun install
bun dev
